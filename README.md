# DeepEmergence.AI
Deep Emergence is a full scale farm management software that allows users to:
   1. Integrate IoT sensors (for unmanaged or semi-managed farming)
   2. Use embedded AI to make smart decisions and monitor overall farm heakth i. realtime.
   3. It will allow integration with several IoT brands, take in data, process to check crop health, presence of diseases and weed and take decisions either by informing or actuating.
